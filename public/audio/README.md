# Audio Folder

Add your audio files here and reference them in the code as:

## Usage Examples:
- `/audio/voice-over-1.mp3`
- `/audio/romantic-message.wav`
- `/audio/personal-recording.m4a`

## Supported Formats:
- MP3 (recommended)
- WAV
- M4A
- OGG

## Tips:
- Keep file sizes reasonable (under 10MB)
- Use compressed audio for faster loading
- Name files descriptively (no spaces, use hyphens)
