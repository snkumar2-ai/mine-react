# Videos Folder

Add your video files here and reference them in the code as:

## Usage Examples:
- `/videos/our-beginning.mp4`
- `/videos/memories.mp4`
- `/videos/proposal.mp4`

## Supported Formats:
- MP4 (recommended)
- WebM
- MOV

## Tips:
- Keep file sizes reasonable for web (under 50MB)
- Use compressed/optimized videos for better loading
- Name files descriptively (no spaces, use hyphens)
